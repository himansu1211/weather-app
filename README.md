# WeatherSphere

A modern, responsive weather application that provides real-time weather information, forecasts, and interactive maps for any location worldwide.

## Features

- **Location Search**: Search for weather information by city, town, or address
- **Current Weather**: Displays current temperature, wind speed, and location details
- **Hourly Forecast**: 24-hour weather forecast with 3-hour intervals
- **5-Day Forecast**: Daily weather predictions with high/low temperatures
- **Interactive Map**: Clickable map powered by Leaflet to select locations
- **Sun & Moon Times**: Sunrise and sunset information
- **UV Index & Updates**: UV safety levels, rain expectations, and weather feedback
- **Geolocation**: Get weather for your current location with one click
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Mapping**: Leaflet.js for interactive maps
- **Weather API**: Open-Meteo (free weather API)
- **Geocoding**: Nominatim (OpenStreetMap geocoding service)
- **Styling**: Custom CSS with dark theme and smooth animations

## Setup and Installation

1. Clone or download the repository
2. Open `index.html` in your web browser
3. No additional setup or dependencies required - it's ready to use!

## Usage

1. **Search by Location**: Enter a city name in the search box and click "Search"
2. **Use Current Location**: Click the location button (üìç) to get weather for your current position
3. **Click on Map**: Click anywhere on the map to get weather for that location
4. **View Forecasts**: Scroll through hourly and daily weather predictions
5. **Additional Info**: Check UV index, rain expectations, and weather feedback in the Updates section

## API Usage

The application uses the following APIs:
- Open-Meteo: For weather data (temperature, wind, forecasts)
- Nominatim: For location search and reverse geocoding

Note: No API keys are required as these services are free and don't need authentication for basic usage.
